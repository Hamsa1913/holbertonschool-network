# holbertonschool-network

## Description

This repository contains networking basics projects for Holberton School.

The project focuses on understanding how computers communicate over networks, including the OSI model, LAN, WAN, the Internet, IP addresses, TCP/UDP protocols, ports, and basic network tools such as `ping` and `netstat`.

## Learning Objectives

By the end of this project, I should be able to explain:

* What the OSI model is
* How many layers the OSI model has
* How the OSI model is organized
* What a LAN is
* What a WAN is
* What the Internet is
* What an IP address is
* The difference between public and private IP addresses
* What localhost means
* What a subnet is
* Why IPv6 was created
* The difference between TCP and UDP
* What a port is
* Common port numbers such as SSH, HTTP, and HTTPS
* How to check if a device is connected to a network using `ping`

## Requirements

* Ubuntu 22.04
* Bash scripts must be executable
* Bash scripts must pass `shellcheck`
* The first line of Bash scripts must be:

```bash
#!/usr/bin/env bash
```

* The second line of Bash scripts must be a comment explaining what the script does
* All files must end with a new line
